# SkillNest

## Project Description
SkillNest is a platform designed to connect learners with skilled professionals for training and mentorship in various fields.

## Installation
To install the project, clone the repository and install the necessary dependencies:

```bash
git clone https://github.com/AnirudhVol2/skillnest.git
cd skillnest
npm install
```

## API Endpoints Documentation
### 1. GET /api/v1/courses
Retrieve a list of all available courses.

### 2. POST /api/v1/enroll
Enroll a user in a course.

### 3. GET /api/v1/users/:id
Retrieve user profile information.

### 4. POST /api/v1/users
Create a new user profile.
